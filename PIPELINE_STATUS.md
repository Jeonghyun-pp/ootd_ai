# 프로젝트 파이프라인 현재 상태

## 현재 실제 동작하는 파이프라인

### 코디 추천 흐름 (현재)
```
사용자 입력 (mood, comment)
  ↓
src/app/ootd/page.tsx의 handleRecommend()
  ↓
setTimeout으로 2초 대기 (로딩 시뮬레이션)
  ↓
Mock 데이터 생성 (하드코딩된 조합)
  ↓
UI에 표시
```

**문제점:**
- ❌ 실제 ML 모델 사용 안 함
- ❌ /api/recommend 호출 안 함
- ❌ ML 서버 연결 안 됨

## 계획된 파이프라인 (아직 미구현)

### 계획 1: ML 서버 사용
```
사용자 입력
  ↓
src/app/ootd/page.tsx → /api/recommend 호출
  ↓
src/app/api/recommend/route.ts → ML 서버 호출
  ↓
ml-server (FastAPI) → 모델 추론
  ↓
결과 반환 → UI 표시
```

**현재 상태:**
- ✅ 코드는 작성됨
- ❌ ML 서버가 실행되지 않음
- ❌ 학습된 모델이 없음
- ❌ 프론트엔드에서 /api/recommend를 호출하지 않음
